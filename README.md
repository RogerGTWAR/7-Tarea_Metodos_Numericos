# Proyecto Métodos Numéricos

Este proyecto implementa diferentes métodos numéricos utilizando Python y Flask, con una interfaz web interactiva.

---

## Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

* Python
* MySQL
* Visual Studio Code
* Extensión de Python para Visual Studio Code

---

## Clonar el repositorio

Abre la consola y ejecuta:

```bash
git clone https://github.com/RogerGTWAR/7-Tarea_Metodos_Numericos.git
```

Luego entra a la carpeta del proyecto:

```bash
cd 7-Tarea_Metodos_Numericos
```

---

## Configuración

* Asegúrate de tener MySQL activo
* Configura la conexión a la base de datos en tu archivo correspondiente (por ejemplo: `ConexionMySQL`)

---

## Ejecutar el proyecto

En la consola, ejecuta:

```bash
python app.py
```

Luego abre tu navegador en:

```text
http://127.0.0.1:5000
```

---

## Funcionalidades

* Método de Bisección
* Falsa Posición
* Punto Fijo
* Newton-Raphson
* Secante
* Métodos numéricos avanzados (Euler, Heun, Runge-Kutta, etc.)
